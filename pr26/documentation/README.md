# Практическая работа №26: Анализ PHP и знакомство с Laravel

## Описание проекта
Анализ legacy PHP-кода и создание современного веб-приложения на фреймворке Laravel.

## Установка и запуск

### Требования
- PHP 8.0+
- Composer
- Расширения PHP: mbstring, xml, ctype, json

### Установка
```bash
composer install
cp .env.example .env
php artisan key:generate
